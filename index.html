Сайт для вашої бібліотеки за один вечір — без сервера, без хостинг-рахунків

Готовий каркас із трьох інформаційних сторінок і розділу новин, який самостійно оновлюється з вашої Google Таблиці. Розгортається на GitHub Pages, редагувати контент може будь-хто без знання коду.

Без сервера

Без власного хостингу

GitHub Pages розміщує статичні сайти без потреби у власному сервері. Не треба ні сервера, ні бази даних, ні системного адміністратора.

Google Таблиця

Новини оновлює будь-хто

Секретар чи бібліотекар додає рядок у знайому Google Таблицю — новина сама з'являється на сайті. Без панелі адміністратора, без пароля.

Готовий каркас

Лишається вписати текст

Структура, стиль і логіка вже готові. Достатньо замінити назву бібліотеки, тексти сторінок і підключити свою таблицю.

Що входить у шаблон

Три статичні інформаційні сторінки (головна, можливості/послуги, детальна інформація) плюс сторінка новин, яка є живим прикладом підвантаження даних із зовнішнього джерела — без бекенду й без плати за API.

i Розділ «Новини» на цьому сайті — не статичний текст. Кожна картка підвантажується напряму з Google Таблиці у форматі CSV. Спробуйте додати рядок у таблицю-джерело — новина з'явиться на сайті без жодного redeploy.

Готові спробувати для своєї бібліотеки?

Скопіюйте структуру шаблону, підключіть власну Google Таблицю замість демонстраційної — і за кілька годин у вашої бібліотеки буде власний сайт з новинами, які оновлюють колеги без допомоги програміста.
