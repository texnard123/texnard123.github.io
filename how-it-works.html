Покроковий запуск

Як розгорнути цей сайт для своєї бібліотеки

Без сервера, без оплати хостингу, без команди розробників — усе робиться через звичайний браузер.

Три кроки до запуску

01

Створіть репозиторій на GitHub

Заведіть акаунт на github.com, створіть репозиторій виду назва.github.io і залийте файли шаблону в корінь — через кнопку «Add file → Upload files», без командного рядка.

02

Підключіть свою Google Таблицю

Створіть таблицю з колонками «Назва», «Опис», «Зображення». Відкрийте доступ «Усі, хто має посилання → Читач» і вставте ID таблиці у файл news.js.

03

Увімкніть GitHub Pages

У Settings → Pages оберіть гілку main і папку /root. Через кілька хвилин сайт буде доступний за адресою назва.github.io.

Щоденне оновлення новин

Після запуску додавати новини може будь-хто з доступом до таблиці — без пояснень про Git чи код. Достатньо додати рядок: назва події, короткий опис і назву файлу картинки.

Саму картинку заливають у папку images/ репозиторію через перетягування файлу в браузері — так само просто, як прикріпити файл до листа.

i Джерело даних для розділу новин на цьому сайті — публічна Google Таблиця, підключена через її CSV-експорт. Це той самий механізм, який запрацює й у вашої бібліотеки — досить замінити ID таблиці на свій.

Інфраструктура (хостинг, сертифікат) не вимагає окремої оплати провайдеру. За бажання можна підключити власний домен замість назва.github.io — це окрема стаття витрат на реєстратора домену, близько 300–500 ₴ на рік залежно від зони.
