# Сайт бібліотеки — інструкція

## Структура
- `index.html` — головна
- `about.html` — про бібліотеку
- `services.html` — послуги та фонди
- `news.html` + `news.js` — новини (тягнуться з Google Таблиці)
- `style.css` — спільний стиль
- `images/` — сюди класти картинки для новин

## Як оновлювати новини
1. Пишеш новину в Google Таблиці: колонки **Назва**, **Опис**, **Зображення**.
2. У колонку "Зображення" вписуєш лише **назву файлу** (наприклад `1.png`), без посилань на Google Drive.
3. Сам файл картинки заливаєш у папку `images/` цього репозиторію на GitHub (кнопка "Add file → Upload files" прямо у вебінтерфейсі GitHub — не треба нічого встановлювати).
4. Сайт сам підтягне нову новину при наступному відкритті сторінки `news.html` — нічого пересобирати не треба.

⚠️ Таблиця має бути відкрита для перегляду за посиланням: **Поділитися → "Усі, хто має посилання" → Читач**.

## Деплой на GitHub Pages
1. Створи репозиторій на GitHub, заливаєш усі ці файли в корінь.
2. Settings → Pages → Source: обираєш гілку `main` і папку `/root`.
3. Через кілька хвилин сайт з'явиться за адресою `https://<твій-юзернейм>.github.io/<назва-репозиторію>/`.
4. Якщо захочеш свій домен — у Settings → Pages додаєш Custom domain і прописуєш DNS у реєстратора домену.

## Якщо треба змінити аркуш таблиці
У `news.js` є змінна `SHEET_GID` — якщо новини на іншому аркуші (не першому), візьми gid із адресного рядка таблиці (число після `gid=`) і встав туди.
